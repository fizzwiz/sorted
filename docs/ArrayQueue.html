Class: ArrayQueue

ArrayQueue(fifoopt, itemsopt)

A basic queue implementation based on an internal array.

In ArrayQueue, newly added items are always appended to the end of the internal array. The extraction order is controlled by a Boolean flag:

  • fifo

When fifo is true, items follow First-In-First-Out (FIFO) behavior; otherwise, they follow LIFO.

Notably, all items are treated as unique: no equivalence or identity checking is used. Even identical items are considered distinct, and:

  • Collection#has
  • Collection#remove

always return false.

The class is optimized for usage of:

  • ArrayQueue#add
  • ArrayQueue#peek
  • ArrayQueue#poll
