Class: Collection

Collection()

Abstract base class for iterable collections.

The Collection class extends Each and defines a common interface for both sorted and unsorted collection types.

Subclasses must implement the following abstract methods:

  • n()
  • has()
  • add()
  • remove()
  • clear()

On top of these, Collection provides concrete utility methods:

  • isEmpty()
  • let()
  • letAll()
  • removeAll()
