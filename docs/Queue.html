Class: Queue

Queue()

The Queue class extends Collection and acts as a template for defining essential methods that every sorted collection should implement:

  • peek()
  • poll()
  • reverse()

It also provides concrete implementations based on these abstract methods, inherited by all subclasses of sorted collections:

  • select()
