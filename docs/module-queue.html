Module: queue

The queue module provides queue-based data structures with different ordering strategies:

  • ArrayQueue: A simple, insertion-ordered queue supporting FIFO or LIFO behavior.
  • SortedArray: A sorted queue maintaining items in comparator-defined order.

These structures implement the Queue interface and support various use cases such as priority queues, bounded collections, or symbolic iteration backends.
